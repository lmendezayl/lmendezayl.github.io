these are some of my public personal projects.

RSI-based trading bot

this is a binance trading bot script that uses the binance api to fetch historical data, calculates the rsi using the ta library, and places buy or sell orders based on rsi thresholds. it loads api keys from environment variables and allows the user to select a cryptocurrency to trade.

exactasdle --wip

exactasdle is a guessing game inspired by Loldle, but focused on the subjects from FCEyN. players must guess the subject by its attributes such as department, program, and modality.
